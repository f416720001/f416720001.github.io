幾個重點

component想要做寫入資料，要透過 commit 去 mutations 呼叫函示來寫入

this.$store.commit('SETDAY', val)

Router

Error 訊息

vue-router.esm.js?8c4f:2062 Uncaught (in promise) Error: Avoided redundant navigation to current location: "/path/1".

解決之道

這裡要特別注意，如果在切換路徑時(帶數字編號那種)，可能會超過，或是數字沒超過但呼叫到router(以下這種)，使得變更路徑重複的問題，建議用if else 把限制條件寫好。

this.$router.replace({
    params: {
        day: day + 1,
    },
});
